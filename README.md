A library for MonoGame.

Implements
  -Graphics
    -Configurable RenderGraph via JSON
  -Input
    -Unity-like InputManager
    -Mouse-to-game coordinates
  -Content
    -AssetManager that auto-loads Content from the MGCB

A 'Testing' project has been inclulded to provide an example of proper order of Initialize, Load, Update, and Render methods; it also has crude graphical tests.
